## [v0.1.0] - 2026-05-09

### Highlights

- Unified incident lifecycle schema spanning premortem, intra-mortem, and postmortem phases
- CLI tool for rendering incident JSON to Markdown with phase-aware template selection

### Added

- JSON Schema for incident lifecycle artifacts with timeline, hypotheses, action items, and evidence ([`4c828b7`](https://github.com/plexusone/incident-lifecycle-spec/commit/4c828b7))
- Go types matching schema with type-safe enums for phase, severity, status, and confidence levels ([`47ad817`](https://github.com/plexusone/incident-lifecycle-spec/commit/47ad817))
- Markdown renderer with embedded templates for intra-mortem and postmortem phases ([`cb201a4`](https://github.com/plexusone/incident-lifecycle-spec/commit/cb201a4))
- View helpers: `ConfirmedFacts()`, `ActiveHypotheses()`, `InProgressActions()` for template filtering ([`cb201a4`](https://github.com/plexusone/incident-lifecycle-spec/commit/cb201a4))
- `ilspec render` command with auto-detection of template based on incident phase ([`e918402`](https://github.com/plexusone/incident-lifecycle-spec/commit/e918402))
- Auto-generated output filenames based on phase (e.g., `incident-update.md` for intra-mortem) ([`b7eafac`](https://github.com/plexusone/incident-lifecycle-spec/commit/b7eafac))
- `--output` flag for explicit output file, use `-` for stdout ([`b7eafac`](https://github.com/plexusone/incident-lifecycle-spec/commit/b7eafac))
- `--template-dir` flag for custom template directories ([`e918402`](https://github.com/plexusone/incident-lifecycle-spec/commit/e918402))

### Documentation

- README with lifecycle model explanation, architecture diagram, and usage instructions ([`99a1ea2`](https://github.com/plexusone/incident-lifecycle-spec/commit/99a1ea2))
- DESIGN_PRINCIPLES covering schema navigability, nesting limits, and schemakit integration ([`99a1ea2`](https://github.com/plexusone/incident-lifecycle-spec/commit/99a1ea2))
- Example intra-mortem incident JSON demonstrating schema usage ([`7bda137`](https://github.com/plexusone/incident-lifecycle-spec/commit/7bda137))
